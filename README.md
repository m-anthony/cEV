Compute your cEV for Betclic, splitted by buy in and/or position

#Installation
---

- download the zip from the latest release : https://github.com/m-anthony/cEV/releases/download/v0.1/cev-0.1.zip
- unzip it
- edit the cev.config.txt file in the "bin" folder so the software can find your hand histories
- launch the cev.bat (for windows) or cev (for linux) and specify the file you want to analyse (or let it empty to scan the whole folder)
